# German Mongolian Anki App

This is a Vite + React wrapper around your uploaded `App.jsx` file.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Netlify

Connect this folder/repo to Netlify, or run `npm run build` and upload the `dist` folder.

## Important note

Right now the app stores data in browser `localStorage`, so user accounts and progress are device-specific until you connect a real backend such as Supabase.
